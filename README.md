# DailyFlow

**DailyFlow – Organize your day and stay on track.**  

DailyFlow is a simple, visual **Task & Habit Tracker** that helps you manage your daily tasks, build healthy habits, and track streaks to stay motivated. Built with **HTML, CSS, and JavaScript**, it uses **LocalStorage** to save your progress so you never lose your data.  

---

## Features 📝

- ✅ Add, remove, and complete tasks easily  
- ✅ Track daily habits with streaks and best performance  
- ✅ Prevent duplicate tasks or habits  
- ✅ Dynamic “No Tasks” / “No Habits” pages  
- ✅ Persistent data using LocalStorage  
- ✅ Tabs to switch between Tasks and Habits seamlessly

## How to Use 🚀

1. **Open the app** in your browser.  
2. **Add Tasks:**  
   - Enter the task name  
   - Choose priority (Easy / Medium / Hard)  
   - Click **Add Task**  
3. **Add Habits:**  
   - Enter the habit name  
   - Click **Add Habit**  
4. **Complete Tasks:**  
   - Click the check icon to mark as completed  
5. **Track Habits:**  
   - Click **Check in Today** to increase streak  

All tasks and habits are saved automatically in your browser using **LocalStorage**.
